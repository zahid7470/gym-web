## 📖 Project Overview

**FORGEHAUS** is a modern, premium, multi-page gym website designed to showcase professional frontend development skills and high-fidelity UI/UX implementation. It is engineered with a dark-first aesthetic, dynamic scroll animations, and a fully responsive grid system to provide a premium user experience across all devices.

Built from a comprehensive Design Requirement Document (DRD), this project demonstrates precision in translating complex visual designs into a functional, production-ready React application. From seamless page transitions to interactive bento grids and accessible accordion components, FORGEHAUS represents the intersection of robust code architecture and elite design execution.

## 💻 Tech Stack

This project was built using modern web development standards to ensure optimal performance, scalability, and maintainability.

*   **Frontend Framework:** React 18
*   **Routing:** React Router v6
*   **Styling:** Vanilla CSS 3 (Custom CSS Properties, Flexbox, CSS Grid)
*   **Animations:** Framer Motion & Intersection Observer API
*   **Icons:** Lucide React
*   **Build Tool:** Vite
*   **Deployment:** Vercel

<br />

## 📂 Project Structure

The project follows a clean, modular architecture separating UI patterns, page views, and core logic.

```
src/
├── components/          # Reusable UI components
│   ├── ui/              # Granular elements (Buttons, Cards, Forms, Accordions)
│   ├── Navbar.jsx       # Global navigation
│   ├── Footer.jsx       # Global footer
│   ├── AnimatedSection.jsx  # Scroll animation wrapper
│   └── PageTransition.jsx   # Route transition wrapper
├── pages/               # Top-level route views
│   ├── Home/            # Landing page (Hero, Stats, Featured areas)
│   ├── About/           # Story, Mission, Facilities using Bento grid
│   ├── Programs/        # Interactive Tabs & Comparison Tables
│   ├── Trainers/        # Dynamic Trainer Grid
│   ├── Membership/      # Pricing Cards & FAQ Accordion
│   ├── Testimonials/    # Success stories in Masonry layout
│   └── Contact/         # Form & Interactive UI
├── index.css            # Global design system (tokens, resets, typography)
├── App.jsx              # Main application shell with routing
└── main.jsx             # React entry point
```

## ✨ Key Features

*   **Premium Dark UI:** Exacting color palette featuring 'Forge Black', 'Ember Amber', and semantic shades for a gritty yet refined look.
*   **Performant Scroll Animations:** Staggered fade-ins and dynamic stat counters utilizing `framer-motion` and `react-intersection-observer`.
*   **Fluid Responsiveness:** Complex layouts (Bento grids, Masonry columns, CSS Grid comparison tables) that gracefully adapt from 4K desktop monitors down to 320px mobile screens.
*   **Accessible Components:** Semantic HTML structure, explicit focus states, and accessible interactive elements like FAQ accordions.
*   **Custom Design System:** Centralized CSS variables ensuring absolute consistency in typography scales, spacing units, and brand colors.
*   **Seamless Routing:** `react-router-dom` integration complete with smooth `AnimatePresence` page transitions and automatic scroll-to-top correction.
